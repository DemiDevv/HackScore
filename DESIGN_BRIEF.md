# HackScore — Дизайн-бриф для Claude Design

## О проекте

**HackScore** — веб-сервис для автоматического учёта и оценки продуктовых решений команд на хакатонах. Три роли: участник, жюри, организатор. Проект для хакатона — нужен максимально профессиональный, запоминающийся дизайн.

## Эстетика и стиль

**Направление:** Modern dashboard / Developer tools aesthetic (как Linear, Vercel, Raycast)

- **Тёмная тема** — основной режим (фон: #0A0A0F, карточки: #12121A, бордеры: #1E1E2E)
- **Акцентный цвет:** электрический индиго (#6366F1) + бирюзовый (#06B6D4) для градиентов
- **Успех/ошибки:** зелёный (#10B981), красный (#EF4444), жёлтый (#F59E0B)
- **Типографика:** заголовки — жирный геометрический sans-serif (например Satoshi, General Sans или Clash Display), тело — чистый sans (DM Sans или Plus Jakarta Sans)
- **Радиусы:** 8-12px для карточек, 6px для кнопок и инпутов
- **Shadows:** subtle glow-эффекты для акцентных элементов (box-shadow с цветом акцента)
- **Иконки:** Lucide Icons — тонкие, минималистичные
- **Анимации:** плавные fade-in, subtle hover-эффекты, skeleton loaders
- **Дифференциатор:** glass-morphism элементы (backdrop-blur) в header и модальных окнах

## Общий Layout

Все роли используют единый layout:

```
┌──────────────────────────────────────────────────────┐
│  HEADER (60px, glass-morphism, backdrop-blur)         │
│  [Logo: HackScore]          [User Avatar + Name ▾]   │
├────────────┬─────────────────────────────────────────┤
│  SIDEBAR   │          MAIN CONTENT                   │
│  (240px)   │                                         │
│            │                                         │
│  Навигация │          Контент страницы               │
│  по ролям  │                                         │
│            │                                         │
│            │                                         │
│            │                                         │
│  ────────  │                                         │
│  [Logout]  │                                         │
└────────────┴─────────────────────────────────────────┘
```

### Header
- Лого слева: иконка + "HackScore" (gradient text индиго → бирюзовый)
- Справа: колокольчик уведомлений (badge), аватар + имя + роль (dropdown: профиль, настройки, выход)
- Стиль: backdrop-blur-xl, border-bottom: 1px solid rgba(255,255,255,0.06)

### Sidebar
- Группы навигации с разделителями и заголовками секций (маленький серый текст)
- Активный пункт: фон акцентного цвета с opacity 15%, текст акцентный, иконка заполненная
- Hover: фон rgba(255,255,255,0.04)
- Внизу: статус хакатона (badge: "В процессе" / "Оценка")
- На мобильных: overlay sidebar с бургер-кнопкой в header

## Дополнительные компоненты UI

### Toast уведомления
- Позиция: bottom-right
- Типы: success (зелёный), error (красный), info (акцентный), warning (жёлтый)
- Анимация: slide-in справа

### Empty states
- Для каждой таблицы — красивый empty state с иллюстрацией
- Текст: "Пока нет команд" + подсказка что делать
- Иконка: большая, приглушённая

### Loading states
- Skeleton loaders для таблиц и карточек
- Spinner для кнопок при отправке
- Progress bar при загрузке файлов

### Модальные окна
- Backdrop-blur фон
- Плавная анимация появления (scale + opacity)
- Кнопка закрытия в правом верхнем углу

### Badges/Tags
- Статусы: pending (серый), running (жёлтый пульсирующий), completed (зелёный), failed (красный)
- Роли: participant (синий), jury (фиолетовый), organizer (оранжевый)
- Вердикты: OK (зелёный), WA (красный), TL (жёлтый), ML (оранжевый), RE (фиолетовый), CE (серый)

## Адаптивность

- **Desktop (>1280px):** полный layout с sidebar
- **Tablet (768-1280px):** sidebar уменьшается до иконок (collapsed mode)
- **Mobile (<768px):** sidebar скрывается, бургер-меню в header, таблицы горизонтально скроллятся, карточки в одну колонку
