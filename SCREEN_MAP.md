# HackScore — Карта экранов и компонентов

## Роли и навигация

### Участник (participant)
```
Sidebar:
├── ХАКАТОН
│   ├── 📋 Мои хакатоны      → HackathonListPage
│   └── 👥 Моя команда        → TeamPage
├── РЕШЕНИЕ
│   ├── 📤 Загрузка решения   → SubmissionPage ⭐ KEY SCREEN
│   └── ✅ Результаты проверок → ResultsPage
└── АЛГОРИТМЫ
    ├── 💻 Задачи              → AlgoTasksPage → AlgoTaskDetailPage
    └── 📊 Мои попытки         → AlgoTasksPage
```

### Жюри (jury)
```
Sidebar:
├── ОЦЕНКА
│   ├── 📋 Решения команд     → ReviewListPage
│   └── ⭐ Мои оценки         → ReviewListPage (filtered)
└── РЕЗУЛЬТАТЫ
    └── 🏆 Лидерборд          → LeaderboardPage
```

### Организатор (organizer)
```
Sidebar:
├── УПРАВЛЕНИЕ
│   ├── 🏆 Хакатон            → HackathonManagePage
│   ├── 📊 Критерии оценки    → CriteriaPage
│   └── 👥 Команды            → TeamsOverviewPage
├── АЛГОРИТМЫ
│   ├── 📝 Задачи             → AlgoManagePage
│   └── 📋 Результаты         → AlgoManagePage
└── ИТОГИ
    ├── 🏆 Лидерборд          → LeaderboardPage
    └── 📥 Экспорт            → LeaderboardPage
```

## Общие экраны

### LoginPage
- Полноэкранный, без sidebar
- Слева (58%): mesh-gradient фон + лого + floating code-блоки
- Справа: форма (email, password) + демо-аккаунты
- Ссылка на регистрацию

### RegisterPage
- Такой же layout как login
- Поля: имя, email, пароль, повтор
- Выбор роли: 3 карточки (participant/jury/organizer)

## Экраны участника

### HackathonListPage
- PageHeader: "Мои хакатоны" + кнопка "Присоединиться к хакатону"
- Grid карточек хакатонов (1-2 колонки)
- Каждая карточка: название, статус badge, команда, дедлайн, прогресс-бар

### TeamPage
- Двухколоночный layout
- Слева: карточка команды + список участников (аватар, имя, email, роль captain/member)
- Справа: форма приглашения по email + список ожидающих

### SubmissionPage ⭐ КЛЮЧЕВОЙ ЭКРАН
- PageHeader со статус-badge (Черновик/Отправлено)
- Stepper: Репозиторий → Документация → Презентация → Скринкаст → Готово
- 4 карточки-секции (вертикальный стек):
  1. **Репозиторий**: radio (URL/Архив) + Input или FileUpload
  2. **Документация**: FileUpload (.pdf/.docx/.md)
  3. **Презентация**: FileUpload (.pptx/.pdf)
  4. **Скринкаст**: radio (Файл/URL) + FileUpload или Input
- Кнопка "Отправить на проверку" (gradient, full-width)
- **После отправки**: блок результатов
  - Grid 2x2 карточек проверок (Код, Документация, Презентация, Скринкаст)
  - Каждая: иконка, статус (spinner/check/error), балл X/10
  - Раскрываемый детальный отчёт при клике:
    - **Код**: структура репозитория (check/x), метрики (LOC, complexity, MI), линтер, безопасность
    - **Документация**: найденные разделы, объём, изображения

### ResultsPage
- Карточка с круговым прогрессом итогового балла (SVG ring)
- Badge с местом в рейтинге
- Таблица критериев: Критерий | Тип (Авто/Эксперт) | Балл | Вес | Взвешенный
- Итоговая строка с суммой

### AlgoTasksPage
- Таблица задач: # | Название | Лимит времени | Лимит памяти | Мой вердикт
- Вердикт-бэйджи: OK (success), WA (error), TL (warning), CE (default)

### AlgoTaskDetailPage
- Кнопка "Назад к задачам"
- Двухколоночный layout (55/45):
  - **Слева**: условие задачи + примеры (input/output в grid)
  - **Справа**: 
    - Редактор кода (textarea + номера строк + выбор языка)
    - Кнопка "Отправить решение"
    - Таблица попыток: #, время, язык, вердикт, ms, MB

## Экраны жюри

### ReviewListPage
- Pill-фильтры: Все / Ожидают оценки / Оценены мной / Частично
- Таблица: # | Команда (аватар + имя) | Статус проверки | Авто-балл | Моя оценка | Действие
- Действие: кнопка "Оценить" или "Изменить"

### ReviewDetailPage ⭐⭐ САМЫЙ ВАЖНЫЙ ЭКРАН
- Навигация между командами (← / →)
- Двухколоночный layout (auto / 380px):
  - **Слева**: Tabs (Код | Документация | Презентация | Скринкаст | Алгоритмы)
    - **Код**: ссылка на репо + авто-оценка + прогресс + структура + метрики + линтер + безопасность
    - **Документация**: авто-оценка + разделы + объём/изображения + скачивание
    - **Презентация**: авто-оценка + кол-во слайдов + разделы (found/missing badges)
    - **Скринкаст**: авто-оценка + метаданные + видео-плеер + транскрипция + AI-саммари
    - **Алгоритмы**: таблица задач (задача, вердикт, попыток, время)
  - **Справа (sticky)**: Панель экспертной оценки
    - Для каждого expert-критерия: ScoreSlider 0-10 + textarea комментарий
    - Кнопка "Сохранить оценки"
    - Статус сохранения (зелёный alert)

## Экраны организатора

### HackathonManagePage
- 4 StatCard в ряд: Команд, Участников, Решений, Жюри
- Карточка статуса хакатона (5 шагов stepper кликабельный)
- Двухколоночный layout:
  - **Слева**: форма настроек (название, описание, даты, макс участников)
  - **Справа**: быстрые действия (4 строки с иконкой + название + описание)

### CriteriaPage
- Индикаторная карточка: всего критериев + сумма весов (зелёный если 100%)
- Кнопка "Добавить критерий"
- Список критериев (sortable cards):
  - Grip icon + название + тип badge (Авто/Эксперт) + описание + вес (крупно) + кнопки edit/trash
  - ProgressBar внизу каждого критерия

### TeamsOverviewPage
- Таблица: # | Команда | Участники (X/5) | Решение (badge) | Авто | Жюри | Статус

### AlgoManagePage
- Таблица задач: # | Название | TL | ML | Тестов | Решено | Действия
- Модальное окно добавления: название, условие (textarea), лимиты, FileUpload для тестов

### LeaderboardPage
- Подиум топ-3 (карточки): 2-е место | 1-е место (выше, glow) | 3-е место
  - Иконка трофея + ранг + имя + балл
  - Цвета: gold, silver, bronze
- Полная таблица: Место | Команда | Авто-оценка | Жюри | Алгоритмы | Итого
  - Топ-3 строки с цветным фоном (medal color 5%)
  - Кнопка "Экспорт"

## Маппинг на Tailwind + shadcn/ui

| Наш компонент | shadcn/ui эквивалент | Примечания |
|---------------|---------------------|-----------|
| Button | `<Button>` | variants: default, secondary, ghost, destructive |
| Badge | `<Badge>` | кастомные variant цвета |
| Card | `<Card>` | hover + glow пропсы |
| Input | `<Input>` | с иконкой слева (composition с Icon) |
| Textarea | `<Textarea>` | |
| Select | `<Select>` | |
| Modal | `<Dialog>` | |
| Tabs | `<Tabs>` | с иконками |
| ProgressBar | `<Progress>` | кастомный цвет |
| Stepper | Кастомный | нет в shadcn — писать свой |
| FileUpload | Кастомный | drag-n-drop zone |
| ScoreSlider | `<Slider>` | + label + value display |
| StatCard | Кастомный | на базе Card |
| Toast | `sonner` или `react-hot-toast` | bottom-right, цветной left-border |
| Avatar | Кастомный | initials + hue from name |
| EmptyState | Кастомный | icon circle + title + description + action |
| PageHeader | Кастомный | title + subtitle + actions slot |
